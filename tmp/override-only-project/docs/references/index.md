# References

Use this directory for durable external context that should be easy to reload locally.

## Good candidates

- Glossary terms or shared vocabulary
- Vendor or protocol notes that matter repeatedly
- Research summaries with dates and links
- Compliance or operational references worth keeping close to the repo

## Rules

- Summarize instead of pasting huge external documents
- Add dates and links when time-sensitive
- Remove or replace references that are no longer trustworthy
